### 5. Solidity tools & frameworks

1. [Zeppelin Solidity](https://github.com/OpenZeppelin/zeppelin-solidity) (development framework)

2. [Truffle](http://truffleframework.com/) (development framework)

3. [Remix Solidity IDE](https://remix.ethereum.org/) (Integrated development environment)

4. [Web3.js](https://github.com/ethereum/web3.js/) (Ethereum Javascript API)

5. [Solium](https://github.com/duaraghav8/Solium) (Analyzes your code for style & security issues)

6. [Javascript TestRPC](https://github.com/trufflesuite/ganache-cli) (local blockchain development)

7. [MetaMask](https://metamask.io/) (browser extension)

8. [Geth](https://www.ethereum.org/cli) (command line interface to run a full Ethereum node)

9. [Solc](https://github.com/ethereum/solc-js) (Solidity compiler)
